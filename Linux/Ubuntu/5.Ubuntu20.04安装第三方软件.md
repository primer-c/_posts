#### 1. 安装第三方软件

- Ubuntu Tree

    ```bash
    sudo apt install tree
    ```

- Ubuntu Git

    ```bash
    sudo apt install git
    ```

- curl

    ```bash
    sudo apt install curl
    ```
- serve静态服务器

    ```bash
    npm install -g serve
    ```
