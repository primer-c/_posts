#### 1. VS Code内置浏览器

- 安装

  ```bash
  Browser Previews
  ```

- 使用

  - 导航栏选择`.html`文件
  - `Ctrl` + `Shift` + `P`打开`Command Palette`窗口
  - 输入： `Browser Previews: Open Active File in Preview`
