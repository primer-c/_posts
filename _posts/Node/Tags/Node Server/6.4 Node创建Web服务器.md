
#### 1. 创建Web服务器

1. 引入系统模块http

  ```js
  // 引用系统模块
  const http = require('http');
  ```

2. 创建web服务器
    
  ```js
  // 创建web服务器
  const app = http.createServer();
  ```

3. 监听端口

  ```js
  // 监听3000端口
  app.listen(3000);
  console.log('服务器已启动，监听3000端口，请访问 localhost:3000')
  ```

4. 响应请求

  ```js
  // 当客户端发送请求的时候
  app.on('request', (req, res) => {
        //  响应
        res.end('<h1>hi, user</h1>');
  });
  ```
