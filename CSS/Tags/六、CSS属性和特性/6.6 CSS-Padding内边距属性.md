#### 1. CSS Padding内边距属性汇总

- 汇总表

  | 属性           | 描述                                                 | CSS  |
  | -------------- | ---------------------------------------------------- | ---- |
  | padding        | 简写属性。作用是在一个声明中设置元素的所内边距属性。 | 1    |
  | padding-bottom | 设置元素的下内边距                                   | 1    |
  | padding-left   | 设置元素的左内边距                                   | 1    |
  | padding-right  | 设置元素的右内边距                                   | 1    |
  | padding-top    | 设置元素的上内边距                                   | 1    |
