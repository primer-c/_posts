#### 1. Var()变量

- 语法： `--变量名`;

- 在body中或者任何想要使用的变量语法的层级中定义var()变量；

- 在任何标签使用；

- 案例

  ```css
  /*定义变量*/
  body {
    --fontSize: 18px;
    --color: #ff0000;
  }

  /*使用变量*/
  .container span {
    font-size: var(--fontSize);
    color: var(--color);
  }
  ``

- HTML

  ```html
  <body>
    <div class="div-1">div-1</div>
    <div class="div-2">div-2</div>
    <div class="div-3">div-3</div>

- CSS

  ```css
  body{
    --width : 200px;
    --height : 200px;
    --color : red;
    --fontSize : 18px;
    --bgColor : #3c3c3c;
  }
  body{
      display: flex;
      justify-content: space-around
  }
  div{
      width: var(--width);
      height: var(--height);
      line-height: var(--height);
      background-color: var(--bgColor);
      text-align: center;

  }
  .div-1{
      color: var(--color);
      font-size: var(--fontSize);
  }
  .div-2{
      color: yellow;
      font-size: calc(var(--fontSize) + 4px);
  }
  .div-3{
      color: green;
      font-size: calc(var(--fontSize) + 8px);
  }
  ```
