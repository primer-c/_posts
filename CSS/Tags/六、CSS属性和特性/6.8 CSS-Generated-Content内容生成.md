#### 1. CSS-Generated-Content内容生成汇总

- 汇总表

  | 属性              | 描述                                                    | CSS  |
  | ----------------- | ------------------------------------------------------- | ---- |
  | content           | 与 :before 以及 :after 伪元素配合使用，来插入生成内容。 | 2    |
  | counter-increment | 递增或递减一个或多个计数器。                            | 2    |
  | counter-reset     | 创建或重置一个或多个计数器。                            | 2    |
  | quotes            | 设置嵌套引用的引号类型。                                | 2    |
