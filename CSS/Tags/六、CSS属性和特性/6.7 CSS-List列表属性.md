####  CSS List列表属性汇总

- 汇总表

  | 属性                | 描述                                       | CSS  |
  | ------------------- | ------------------------------------------ | ---- |
  | list-style          | 简写属性。在一个声明中设置所有的列表属性。 | 1    |
  | list-style-image    | 将图象设置为列表项标记。                   | 1    |
  | list-style-position | 设置列表项标记的放置位置。                 | 1    |
  | list-style-type     | 设置列表项标记的类型。                     | 1    |
  | marker-offset       |                                            | 2    |
