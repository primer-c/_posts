#### 1. CSS Table表格属性汇总

- 汇总表

| 属性            | 描述                                         | CSS  |
| --------------- | -------------------------------------------- | ---- |
| border-collapse | 规定是否合并表格边框。                       | 2    |
| border-spacing  | 规定相邻单元格边框之间的距离。               | 2    |
| caption-side    | 规定表格标题的位置。                         | 2    |
| empty-cells     | 规定是否显示表格中的空单元格上的边框和背景。 | 2    |
| table-layout    | 设置用于表格的布局算法。                     | 2    |
