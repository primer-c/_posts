#### 1. CSS Margin外边距属性汇总

- 汇总表

  | 属性          | 描述                                       | CSS  |
  | ------------- | ------------------------------------------ | ---- |
  | margin        | 简写属性。在一个声明中设置所有外边距属性。 | 1    |
  | margin-bottom | 设置元素的下外边距。                       | 1    |
  | margin-left   | 设置元素的左外边距。                       | 1    |
  | margin-right  | 设置元素的右外边距。                       | 1    |
  | margin-top    | 设置元素的上外边距。                       | 1    |
