- 计算单位：

  ```html
  p { width: calc(80% -20px)}
  ```

- 角度

  |单位标识符|说明|
  |deg|角度：取值 0 ~ 360 deg|
  |grad|百分度：取值 0 ~ 400 grad|
  |rag|弧度 : 取值 0 ～ 6.28rad|
  |turn|圆周：1turn = 360 deg|

- 时间

  |单位标识符|说明|
  |s|秒：1s = 1000ms|
  |ms|毫秒：1000ms=1s|
