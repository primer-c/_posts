#### 1. 一键格式化插件

- 格式化代码： HTML CSS JS Less Sass TypeScript 等

  ```bash
  Prettier - Code formatter
  ```

- 配置： settings

  ```json
  {
    "editor.formatOnSave": true
  }
  ```
