#### 1.HTML Boilerplate

- 使用HTML模版插件，你就摆脱了为HTML新文件重新编写头部和正文标签的苦恼；

- 你只需在空文件中输入html，并按Tab键，即可生成干净的文档结构

  ```bash
  HTML Boilerplate
  ```
