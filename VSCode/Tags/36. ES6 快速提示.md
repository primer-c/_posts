#### 1.JavaScript(ES6) code snippets

- snippet： 指能够帮助快速输入重复代码模板

  ```bash
  JavaScript(ES6) code snippets
  ```
- 方法一

    - 通过快捷键「Ctrl + Shift + P」打开命令窗口（All Command Window）

    - 输入「snippet」，点选「Preferences：Configure User Snippets」

- 方法二

- 点击左边栏： Setting

- User Snippets
