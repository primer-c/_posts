#### 2.常用元素

1. 空元素

   没有内容的元素

   ```html
   I like
   <div></div>
   apples and oranges
   ```

2. 自闭合元素

   对于空元素，可以用一个标签代替，以简洁标签，称为自闭合标签；

   ```html
   I like
   <div />
   apples and oranges
   ```

3. 虚元素：void element

   ```html
   I like apples and oranges
   <hr />
   Today was warm and sunny <br />
   I would like go fishing;
   ```
