- HTML 实体： HTML 预留字符

- 在 HTML 中不能使用小于号（<）和大于号（>），这是因为浏览器会误认为它们是标签；

- 如果希望正确地显示预留字符，我们必须在 HTML 源代码中使用字符实体（character entities）

- 字符实体类似这样：

  ```html
  <div>&lt;</div>

  <!-- or -->

  <div>&#60;</div>
  ```

- 表一： HTML 实体

  | 字符 | 实体名称 | 实体编码 |
  | :--: | :------: | :------: |
  | ' '  |  &nbsp;  |  &#160;  |
  |  <   |   &lt;   |  &#60;   |
  |  >   |   &gt;   |  &#62;   |
  |  &   |  &amp;   |  &#38;   |
  |  "   |  &quot;  |  &#34;   |
  |  ￠  |  &cent;  |  &#162;  |
  |  '   |  &apos;  |  &#39;   |
  |  €   |  &euro;  | &#8364;  |
  |  £   | &pound;  |  &#163;  |
  |  ¥   |  &yen;   |  &#165;  |
  |  ©   |  &copy;  |  &#169;  |
  |  ®   |  &reg;   |  &#174;  |
  |  ™   | &trade;  | &#8482;  |
  |  ×   | &times;  |  &#215;  |
  |  ÷   | &divide; |  &#247;  |
