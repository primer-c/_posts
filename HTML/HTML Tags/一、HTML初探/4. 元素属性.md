#### 1.元素属性(attribute)

1. 一个元素可以有多个属性

   ```html
   I like <a class="link" href="/apple.html" id="firstlink">apples</a> and
   oranges
   ```

2. 使用布尔属性

   ```html
   Enter your name <input disabled />
   ```

3. 自定义属性

   ```html
   Enter your name
   <input disabled="true" data-creator="adam" data-purpose="collection" />
   ```
